# unite-go-imports

## Requirements

- `go get github.com/ToQoz/goimps`
- Add `set rtp+=$GOPATH/src/github.com/ToQoz/goimps/misc/vim` to your vimrc

## Installation

### with NeoBundle

Add `NeoBundle 'ToQoz/unite-go-imports'`

### with Vundle

Add `Bundle 'ToQoz/unite-go-imports'`

## Usage

```
:Unite go/import
:Unite go/drop
```
